Detailed Description

Class containing Skinned Mesh SDF.

Skinned Mesh SDF is SDF generated from static mesh. It's implemented as NeuralSDF per bone.

You need to generate Skinned Mesh SDF before use. That can only be done in editor, so you need to generate all meshes you intend to use with Neural SDF. For generation see GenerationQueue.
