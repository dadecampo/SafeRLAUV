Zibra Effects API Reference

API Reference provides details of C# interface for Zibra Effects.
To efficiently use API, you should familiarize yourself with basic concepts of Zibra Effects, so if you haven't already, please check out User Guide.

To start browsing API Reference, go to Classes list to find one you are interested in.
If you aren't sure where to start from:

  • For Liquids, go to com.zibra.liquid.Solver.ZibraLiquid, which is simulation volume component, responsible for running liquid simulation
  • For Smoke & Fire, go to com.zibra.smoke_and_fire.Solver.ZibraSmokeAndFire which is simulation volume component, responsible for running smoke & fire simulation

Points of interest for Liquids would be:

  • com.zibra.liquid.Manipulators - Namespace with all manipulator types.
  • com.zibra.liquid.SDFObjects - Namespace with all SDF types.
  • com.zibra.liquid.DataStructures - Namespace with classes that serve as parameter containers for liquid.

Points of interest for Smoke & Fire would be:

  • com.zibra.smoke_and_fire.Manipulators - Namespace with all manipulator types.
  • com.zibra.smoke_and_fire.SDFObjects - Namespace with all SDF types.
  • com.zibra.smoke_and_fire.DataStructures - Namespace with classes that serve as parameter containers for liquid.
