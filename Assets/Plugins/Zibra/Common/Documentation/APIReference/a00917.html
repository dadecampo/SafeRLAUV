Detailed Description

Detector for liquid particles.

Calculates number of particles inside its shape.

Updated each simulation step.

Property Documentation

◆ ParticlesInside

int ParticlesInside = 0
get

Number of particles inside detector.

Same number of particles can correspond to different volume, depending on liquid settings.

Since liquid is somewhat compressible, even inside same simulation, same number of particles can have somewhat different volume.

◆ BoundingBoxMin

Vector3 BoundingBoxMin = new Vector3(0.0f, 0.0f, 0.0f)
get

Bottom left corner of 3D bounding box of detected liquid (in world coordinates).

If no liquid was detected, this field will be set to zeros and the whole bounding box area will equal 0.

(Editor only) Event that is triggered when state of manipulator changes to trigger update of custom editor.
