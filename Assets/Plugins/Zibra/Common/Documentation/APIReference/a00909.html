Detailed Description

Colliders for the liquid.

You can't use Unity's colliders with the liquid, you can only use this specific collider component with liquid.

Each collider needs to be added to list of collider in liquid. Otherwise it won't do anything. This is needed since you may have multiple liquids with separate colliders.
