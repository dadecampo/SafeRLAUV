Detailed Description

Base class for liquid manipulator.

It doesn't execute anything by itself, it is used by ZibraLiquid instead.

Each manipulator needs to have shape. Shape is definited by adding SDF component to same GameObject.

Do not try to make custom manipulator. Each and every type of manipulator is heavily tied to logic inside the native plugin. And since you can't change native plugin, you can't add manipulators.

Each manipulator needs to be added to list of manipulators in liquid. Otherwise it won't do anything. This is needed since you may have multiple liquids with separate manipulators.
